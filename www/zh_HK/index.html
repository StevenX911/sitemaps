什麼是 Sitemap?

Sitemap 可讓網站管理員易於通知搜尋引擎關於網站中有哪些可供檢索的網頁。最簡單的 Sitemap 形式就是將網站 URL 及與各 URL 相關的其他中繼資料 (上次更新日期、變更頻率，以及相對於網站中其他 URL 的重要性) 列在 XML 檔中，讓搜尋引擎能夠更有效地檢索該網站。

網頁檢索器通常會發現來自網站內的連結與來自其他網站之網頁。Sitemaps 會補充此資料，讓支援 Sitemaps 的檢索器擷取 Sitemap 中的所有 URL，並使用相關的中繼資料來取得關於這些 URL 的資訊。使用 Sitemap 通訊協定並不保證網頁會出現在搜尋引擎中，但能夠協助網頁檢索器更完整地檢索您的網站。

Sitemap 0.90 是根據 Attribution-ShareAlike Creative Commons License 的條款提供；它的支援範圍極廣，包含來自 Google、Yahoo! 和 Microsoft 的支援。
